# AI Customer Support Agent

## Overview

Automatically classifies incoming emails into:

- Quote Request
- Shipment Tracking
- General Inquiry
- Technical Support

## Technologies

- n8n
- OpenAI GPT-5
- Gmail
- PostgreSQL
- Slack
- Supabase

## Workflow

Email Received

↓

AI Classification

↓

Save to Database

↓

Notify Support Team

↓

Auto Reply Customer
